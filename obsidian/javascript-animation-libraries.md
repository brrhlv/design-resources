# Javascript Animation Libraries

JavaScript libraries for animations and motion effects.

## Resources

- [GSAP](https://gsap.com/) - A JavaScript library for building high-performance animations that work in every major browser
- [Velocity.js](http://velocityjs.org/) - Lightweight animation engine with the same API as jQuery's $.animate()
- [lax.js](https://github.com/alexfoxy/laxxx) - Smooth, lightweight vanilla JavaScript plugin for scroll-triggered animations
- [Rellax.js](https://github.com/dixonandmoe/rellax) - Buttery smooth, super lightweight vanilla JavaScript parallax library
- [three.js](https://github.com/mrdoob/three.js/) - Easy to use, lightweight 3D library with a default WebGL renderer
- [Atropos](https://atroposjs.com/) - Lightweight library for creating 3D parallax hover effects on touch-friendly interfaces
- [wow.js](https://wowjs.uk/) - Reveal animations when you scroll with this easy-to-use library
- [chocolat.js](http://chocolat.insipi.de/) - Free lightbox plugin for displaying images in responsive galleries
- [Tilt.js](http://gijsroge.github.io/tilt.js/) - Lightweight parallax hover tilt effect powered by requestAnimationFrame
- [Roughnotation](https://roughnotation.com/) - Small JavaScript library to create and animate annotations on a web page
- [tsParticles](https://particles.js.org/) - Lightweight library for creating particles, improved version of particles.js
- [Particles.js](https://vincentgarreau.com/particles.js/) - Lightweight JavaScript library for creating particle backgrounds
- [mo.js](https://mojs.github.io/) - The motion graphics toolbelt for the web
- [Lightbox2](https://lokeshdhakar.com/projects/lightbox2/) - Small library to overlay images on top of the current page
- [Slick](https://kenwheeler.github.io/slick/) - Fully responsive carousel with multiple configurations
- [barba.js](https://barba.js.org/) - Create fluid, smooth transitions between website pages
- [Locomotive Scroll](https://locomotivemtl.github.io/locomotive-scroll/) - Scroll library with element detection and smooth parallax effects
- [Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/) - Free responsive jQuery carousel with touch support
- [Swiper](https://swiperjs.com/) - Free, open source modern slider without jQuery with framework support
- [Splide](https://splidejs.com/) - Free, pure JS library for carousels and sliders with no dependencies
- [Simple Parallax](https://simpleparallax.com/) - Easiest approach to implementing parallax effects with JavaScript
- [KUTE.js](https://thednp.github.io/kute.js/) - JavaScript animation engine for modern browsers
- [Granim.js](https://sarcadass.github.io/granim.js/index.html) - Create fluid, interactive gradient animations with this small library
- [Popmotion](https://popmotion.io/) - Simple animation libraries for creating delightful user interfaces
- [Vivus](https://maxwellito.github.io/vivus/) - Library to animate SVGs with a drawn appearance effect
- [Typed.js](https://mattboldt.com/demos/typed-js/) - A JavaScript typing animation library
- [Progress Bar.js](https://kimmobrunfeldt.github.io/progressbar.js/) - Responsive progress bars with animated SVG paths
- [Midnight.js](https://aerolab.github.io/midnight.js/) - Switch fixed headers on the fly as you scroll
- [Theatre.js](https://www.theatrejs.com/) - Animation library with GUI for DOM, WebGL, and JavaScript variables
- [nice-waves](https://gvguy.github.io/nice-waves/) - Beautiful animated waves for your website backgrounds
- [Vanta.js](https://www.vantajs.com/) - Animated website backgrounds in a few lines of code
- [Splitting.js](https://splitting.js.org/) - Creates elements and CSS variables to enable text and grid animations
