# Javascript Chart Libraries

JavaScript libraries for data visualization and charts.

## Resources

- [Chart.js](https://www.chartjs.org/) - Simple yet flexible JavaScript charting for designers and developers
- [Frappe Charts](https://frappe.io/charts) - GitHub-style grid charts and additional visualization options
- [G2Plot](https://g2plot.antv.vision/en) - Interactive, responsive charting library based on grammar of graphics
- [Google Charts](https://developers.google.com/chart/) - Create chart visualizations similar to those Google uses
- [D3.js](https://d3js.org/) - Bind complex data to interactive graphical representations
- [ApexCharts](https://apexcharts.com/) - Modern and interactive open-source charts library
- [Chartist](http://gionkunz.github.io/chartist-js/index.html) - Provides simple, responsive charting solutions
- [Vizzu](https://lib.vizzuhq.com/0.3.0/) - Free, open-source library for creating animated charts and data stories
- [Ant Design Charts](https://ant-design-charts.antgroup.com/en/) - Dynamic data visualization with excellent documentation
- [Nivo](https://nivo.rocks/) - React components for data visualization built on D3
- [ECharts](https://echarts.apache.org/en/index.html) - Open source JavaScript visualization library with declarative syntax
- [VChart](https://visactor.io/vchart) - Multi-terminal charting library for rapid deployment
- [VTable](https://visactor.io/vtable) - High-performance multidimensional analysis component using Canvas
- [VStory](https://visactor.io/vstory) - Narrative visualization system with built-in components
