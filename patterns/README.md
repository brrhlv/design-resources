# Pattern Libraries & Component Collections

Local copies of high-quality design patterns and component libraries for reference and inspiration.

## Contents

| Library | Stars | Description | Key Features |
|---------|-------|-------------|--------------|
| **react-bits** | 6.3k+ | Animated React components | 110+ components in 4 categories: TextAnimations, Animations, Components, Backgrounds. Supports JS/TS with CSS/Tailwind variants |
| **advanced-react-patterns** | 12k+ | Kent C. Dodds workshop | Compound Components, Flexible Compound Components, Prop Collections/Getters, State Reducers, Control Props |
| **simorgh** | 800+ | BBC's React design system | Large-scale production components, contexts, hooks, utilities |
| **animated-react-collection** | 400+ | Curated animated React list | Meta-list linking to best animated component libraries |
| **hyperui** | 11.8k | Tailwind CSS v4 components | 600+ free components for marketing, eCommerce, application UI |
| **flowbite** | 9k | Tailwind UI framework | Complete component library with Figma design files |
| **nostyle** | 161 | Accessible design system | A11y-focused forms and components by Adam Silver |

## Usage

These are reference implementations. To use components:

1. Browse the source files for patterns you need
2. Copy and adapt relevant code to your project
3. Check each library's license for usage terms

## React Patterns (react-bits)

Component variants available:
- **JS-CSS**: JavaScript with plain CSS
- **JS-TW**: JavaScript with Tailwind CSS
- **TS-CSS**: TypeScript with plain CSS
- **TS-TW**: TypeScript with Tailwind CSS

Categories:
- `TextAnimations/` - Animated text effects
- `Animations/` - Motion components
- `Components/` - Interactive UI elements
- `Backgrounds/` - Animated backgrounds

## Advanced React Patterns

Workshop patterns:
1. **Compound Components** - Components that work together
2. **Flexible Compound Components** - Using context for flexibility
3. **Prop Collections & Getters** - Reusable prop patterns
4. **State Reducers** - Inversion of control for state
5. **Control Props** - Controlled component pattern
