# CSS Methodologies

CSS methodologies to help write modular, reusable and scalable code.

## Resources

- [OOCSS](http://oocss.org/) - Object-Oriented CSS concepts for flexible, modular and interchangeable components
- [Atomic CSS](https://acss.io/) - CSS architecture favoring small, single-purpose classes with names based on visual function
- [BEM](http://getbem.com/) - Block Element Modifier methodology for reusable components and code sharing
- [SMACSS](http://smacss.com/) - Scalable and Modular Architecture for CSS - a flexible design process framework
- [CUBE CSS](https://cube.fyi/) - CSS methodology orientated towards simplicity, pragmatism and consistency
