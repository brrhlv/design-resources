# Svelte UI Libraries

UI component libraries for Svelte.

## Resources

- [Svelte Material UI](https://sveltematerialui.com/) - UI library for Svelte based on Material Design
- [SvelteStrap](https://bestguy.github.io/sveltestrap/) - UI library for Svelte based on the Bootstrap framework
- [Svelte Flat UI](https://svelteui.js.org/) - UI library for Svelte based on Flat Design
- [Attractions](https://illright.github.io/attractions/) - A pretty cool UI kit for Svelte
- [Svelteit](https://docs.svelteit.dev) - Minimalistic UI/UX component library for Svelte and Sapper projects
- [Carbon Components Svelte](https://carbon-components-svelte.onrender.com/) - Component library implementing IBM's Carbon Design System
- [Radix Svelte](https://www.radix-svelte.com/) - Port of Radix UI for Svelte
- [shadcn-svelte](https://www.shadcn-svelte.com/) - Unofficial community-led Svelte port of shadcn/ui
- [Skeleton](https://www.skeleton.dev/) - Open source UI toolkit built with Svelte and Tailwind CSS
