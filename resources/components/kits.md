# UI Components & Kits

UI component libraries and design kits for building modern web applications.

## Resources

- [UILang](http://uilang.com/) - A minimal, UI-focused programming language for web designers
- [Medialoot CSS Components](https://medialoot.com/free-themes/css-components/) - Calendars, price grids and other UI components
- [Froala Design Blocks](https://froala.com/design-blocks) - Over 170 responsive design blocks ready for web or mobile apps
- [Material Design For Bootstrap](https://fezvrasta.github.io/bootstrap-material-design/) - Open source toolkit for building material design with Bootstrap
- [Photonkit](http://photonkit.com/) - Desktop UI library for Electron
- [Flat UI](https://designmodo.github.io/Flat-UI/) - Minimal free user interface kit
- [Shards](https://designrevision.com/downloads/shards/) - Modern UI toolkit for web makers based on Bootstrap
- [Creative Tim](https://www.creative-tim.com/) - UI libraries and kits including JS frameworks like React
- [Brumm Shadow Maker](https://brumm.af/shadows) - Online tool to make CSS shadows
- [SpinKit](https://tobiasahlin.com/spinkit/) - Simple CSS Spinners
- [Moving Letters](https://tobiasahlin.com/moving-letters/) - Animated Text with JavaScript and anime.js
- [CSS Layout](https://csslayout.io/) - Popular web layouts and patterns in pure CSS
- [Codyhouse](https://codyhouse.co/) - Front-end framework with accessible HTML, CSS, JavaScript components
- [Tailwind Starter Kit](https://github.com/creativetimofficial/tailwind-starter-kit) - Beautiful extension for TailwindCSS
- [Tailwindtoolbox](https://www.tailwindtoolbox.com/) - Starter templates, components, and utilities for Tailwind CSS
- [tailwindcomponents](https://tailwindcomponents.com/) - Free community components using TailwindCSS
- [Material-UI](https://material-ui.com/) - React components for faster and easier web development
- [Chakra UI](https://chakra-ui.com/) - Simple, accessible component library for React
- [Ant Design](https://ant.design/) - Enterprise-class UI design system
- [Semantic UI](https://semantic-ui.com/) - User-friendly component framework
- [Storybook](https://storybook.js.org/) - Development environment for UI components
- [shadcn/ui](https://ui.shadcn.com/) - High-quality React components you can copy/paste
- [Headless UI](https://headlessui.com/) - Unstyled accessible components for React/Vue
- [Daisy UI](https://daisyui.com/) - Component library built on Tailwind CSS
- [Mantine](https://mantine.dev/) - React components and hooks library with TypeScript support
- [NextUI](https://nextui.org/) - Modern React UI library with beautiful defaults
- [UIkit](https://getuikit.com/) - Lightweight and modular front-end framework
- [Evergreen](https://evergreen.segment.com/) - React UI framework by Segment
- [Grommet](https://v2.grommet.io/) - React-based framework for building responsive apps
- [Rebass](https://rebassjs.org/) - Primitive React UI components using styled-system
- [Blueprint](https://blueprintjs.com/) - React-based UI toolkit for web applications
- [Fluent UI](https://developer.microsoft.com/en-us/fluentui) - Microsoft's design system and components
