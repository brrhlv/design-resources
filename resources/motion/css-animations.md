# CSS Animations

CSS animation libraries and tools.

## Resources

- [Animate.css](https://animate.style/) - Just-add-water CSS animations library with cross-browser compatibility
- [Anime.js](https://animejs.com/) - Lightweight JavaScript animation library with a simple, yet powerful API
- [Magic Animations](https://www.minimamente.com/project/magic/) - CSS3 Animations with special effects
- [Zdog](https://zzz.dog/) - Round, flat, designer-friendly pseudo-3D engine for canvas and SVG
- [CSShake](http://elrumordelaluz.github.io/csshake/) - CSS classes to move your DOM with various shake effects
- [Hover.css](http://ianlunn.github.io/Hover/) - Collection of CSS3 powered hover effects for links, buttons, logos, and more
- [AniJS](http://anijs.github.io/) - Declarative handling library for CSS animations
- [Animista](http://animista.net/) - CSS animation playground with ready to use animations
- [Bounce.js](http://bouncejs.com/) - Tool and JS library for generating beautiful CSS3 powered keyframe animations
- [Tachyons-animate](https://github.com/anater/tachyons-animate) - Single-purpose classes to help you orchestrate CSS animations
- [Sequence.js](https://www.sequencejs.com/) - CSS animation framework for creating responsive sliders, presentations, and banners
- [Infinite](https://tilomitra.github.io/infinite/) - Pure CSS animations that you can drop into your projects
- [OBNOXIOUS.CSS](http://tholman.com/obnoxious/) - Animations for the strong of heart and weak of mind
- [MOTION UI](https://zurb.com/playground/motion-ui) - Sass library for creating transitions and animations
- [Thoughtbot CSS Animation Guide](https://thoughtbot.com/blog/transitions-and-transforms) - Guide to CSS transitions and transforms
- [SVG Artista](https://svgartista.net/) - SVG drawing animation generator
- [AnimXYZ](https://animxyz.com/) - Composable CSS animation toolkit with multiple animation effects
- [Whirl](https://github.com/jh3y/whirl) - CSS loading animations with minimal effort
- [Hamburgers](https://jonsuh.com/hamburgers/) - Tasty CSS-animated hamburger menu icons
- [CSS Loaders & Spinners](https://cssloaders.github.io/) - Collection of different types of CSS loaders and spinners
- [Motion One](https://motion.dev/) - A new animation library built on the Web Animations API
- [pocoloco](https://pocoloco.io/) - Generate dynamic backgrounds for your website
- [AniX](https://adajuly.github.io/AniX/) - Super easy and lightweight animation library
- [AOS](https://michalsnik.github.io/aos/) - Animate On Scroll library for scroll-triggered animations
- [Animatopy](https://sarthology.github.io/Animatopy/) - Just-add-water CSS animations snippets for animations
- [Gradienty Animations](https://gradienty.codes/animations) - Collection of CSS gradient animations
