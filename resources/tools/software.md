# Downloadable Design Software

Desktop design applications and software.

## Resources

- [Gimp](https://www.gimp.org/) - Free & open source image editor similar to Photoshop
- [Gravit Designer](https://www.designer.io/en/) - Free full-featured vector graphic design app for all platforms
- [Blender](https://www.blender.org/download/) - Open source, free animation, 3D modeling, and more
- [Raw Therapee](https://rawtherapee.com/) - Cross-platform raw image processing program
- [Be Funky](https://www.befunky.com/features/graphic-designer/) - Online design program with graphic design features
- [Krita](https://krita.org/en/download/krita-desktop/) - Sketching and painting program for digital artists
- [Inkscape](https://inkscape.org/) - Powerful free vector design tool
- [Adobe XD](https://www.adobe.com/products/xd.html) - Free design tool from Adobe for UI/UX
- [Shapes.so](https://shape.so/pricing) - Icons that can be used as code in your projects
- [Lunacy](https://icons8.com/lunacy) - Sketch alternative for Windows
- [InVision Studio](https://www.invisionapp.com/studio) - Free screen designing tool from InVision
- [Darktable](https://www.darktable.org/) - Free & open source photography workflow and raw developer
- [Inpixio Photo Editor](https://www.inpixio.com/free-photo-editor) - Free photo editor for Windows
- [Colorpicker](https://colorpicker.fr) - Open source colors software for retrieving and storing colors
- [Google Web Designer](https://webdesigner.withgoogle.com/) - Create interactive HTML5 designs and motion graphics
- [Origami Studio](https://origami.design) - Interactive interface design tool by Facebook (Mac only)
